

## 获取YouTube Key
https://console.developers.google.com/apis/api/youtube.googleapis.com
- 1) YouTube Data API v3 > 启用
- 2) 凭据 > 创建凭据 > API密钥
- 3) OAuth 同意屏幕 > 创建应用
_ 4) 凭据 > 创建凭据 > OAuth客户端

## 解绑
https://myaccount.google.com/permissions
