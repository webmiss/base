## 开关
```html
<wm-switch></wm-switch>
```
- value: {type: Boolean, default: false},             //值
- activeColor: {type: String, default: '#6FB737'},    //打开颜色
- inactiveColor: {type: String, default: '#DCDFE6'},  //关闭颜色
- @update:value                                       //事件
