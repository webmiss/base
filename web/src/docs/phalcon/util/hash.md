## 引入
```php
use Util\Hash;
```

## Md5
```php
Hash::Md5(string $data);
```

## Sha256
```php
Hash::Sha256(string $data);
```

## HmacSha1
```php
Hash::HmacSha1(string $data, string $key, bool $binary=true);
```

## HmacSha256
```php
Hash::HmacSha256(string $data, string $key, bool $binary=true);
```
