# WebMIS-Flutter
采用Flutter开发，包括APP自动更新、Socket通信、底部导航菜单、PageView页面布局、Scroller滑动、Popup弹出层等功能。

<br/>

## 一、下载
```bash
git clone https://github.com/webmiss/base.git --depth=1
```
- WebMIS-Base > flutter

<br/>

## 二、运行
#### Bash命令
```bash
# 依赖包
./bash install
# 运行
./bash serve
```

#### Cmd命令
```bash
# 依赖包
.\cmd.bat install
# 运行
.\cmd.bat serve
```
