## 引入
```python
from util.hash import Hash
```

## Md5
```python
Hash.Md5(data: str)
```

## Sha256
```python
Hash.Sha256(data: str)
```

## HmacSha1
```python
Hash.HmacSha1(data: str, key: bytes)
```

## HmacSha256
```python
Hash.HmacSha256(data: str, key: bytes)
```

## Bytes转为16进制
```python
Hash.HmacSha256(byte: bytes)
```
